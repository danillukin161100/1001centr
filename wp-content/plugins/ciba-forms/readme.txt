После установки и активации плагина необходимо в разделе "Инструменты" -> "Ciba Forms" заполнить все необходимые поля

Далее в самих формах ВМЕСТО всех атрибутов выводим результат функции cibaFormsAttributes(...$classes). Пример:
<form <?= cibaFormsAttributes('repair-form') ?>>

Поле с номером телефона обязательно должен иметь атрибут name="phone",
также автоматически будет вставлена маска ввода ко всем тегам input с таким атрибутом

Список дополнительных полей на случай, если необходимо в конкретной форме изменить значение поля по умолчанию из админки:
1. source_id;
2. mango_name.
Например:
<input type="hidden" name="source_id" value="12345">

Пример добавления функций обработки отправки в JS:
jQuery(document).ready(($) => {
	window.CibaForms.success = ($form) => {
		alert('Успех');
	};
	window.CibaForms.error = ($form) => {
		alert('Ошибка');
	};
});
